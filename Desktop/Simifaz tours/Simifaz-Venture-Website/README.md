# Simifaz Venture Website

## Project Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Local Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Accessing the Site

After running `npm run dev`, you will see a URL in your terminal (usually `http://localhost:5173`). Open this URL in your browser to view the website.
